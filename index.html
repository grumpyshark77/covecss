About Cove CSS

Cove CSS is my personal development studio — where I build focused, well-crafted browser tools for developers, designers, and educators.

My work sits at the intersection of web standards, accessibility, and education. Every project I ship is built to help someone understand something better — whether that's a designer understanding colour contrast, or a developer learning how real users experience the web.

I work in vanilla JavaScript, Chrome Extension APIs, and CSS — keeping dependencies minimal and performance tight.
